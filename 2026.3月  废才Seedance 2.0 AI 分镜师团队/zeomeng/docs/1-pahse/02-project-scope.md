# 项目范围

## 1. 本期范围内

- 人工首次登录即梦网页
- 保存并复用登录态
- 自动找到图片生成入口
- 自动输入提示词并点击生成
- 自动等待图片生成完成
- 自动下载所有生成图片
- 自动完成图片重命名与映射记录
- 输出执行日志与错误日志

## 2. 本期范围外

- 图片评分
- 图片筛选
- 前后帧管理
- 视频生成
- 自动注册账号或自动输入账号密码登录
- 绕过验证码、风控、限流

## 3. 人工与系统分工

### 人工负责

- 登录
- 验证码与风控处理
- 准备提示词
- 抽样检查下载结果

### 系统负责

- 复用登录态
- 定位图片生成页
- 输入提示词并提交
- 等待完成、下载、命名和记日志

## 4. 交付边界

本期交付的是文档方案和后续开发依据，目标是为“一期图片自动化闭环”建立明确边界，不提前混入二期需求。

## 5. 技术选型与部署边界

### 技术选型

- 浏览器执行框架：`OpenClaw`
- 浏览器能力载体：`browser skill`
- 本地后处理：本地文件处理脚本

### 部署边界

- 本期文档明确采用 OpenClaw 作为目标框架
- 但当前电脑不承担 OpenClaw 部署
- 因此本期交付不包含 OpenClaw 安装、服务启动和本机常驻运行
- 当前阶段只约束后续实施时必须兼容 OpenClaw browser skill 的调用方式
