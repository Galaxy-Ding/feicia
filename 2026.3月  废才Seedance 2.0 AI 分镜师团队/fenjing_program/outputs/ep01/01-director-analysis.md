# 导演讲戏本

## P01 惊讯落台

- 人物：顾星澜、太史局同僚
- 场景：司天台
- 镜头组：单镜头
- 时长建议：8s

**导演阐述**：  
寅时将尽，司天台上天光未亮，冷青色晨雾贴着石阶浮动，顾星澜独自伏在案前校核春分节气时刻，指尖拨动算筹，动作快而稳，带着多年练就的克制与专注。忽然，消息已传至太史局——沈大人被带走了。这个消息并不需要被谁当面喊出，它像一阵骤然蔓开的寒气，先一步抵达高台，也瞬间刺破原本静止的工作场域。顾星澜整个人像被闷响击中，手中算筹骤然失力，珠子噼啪落地，沿着石阶一路滚下，清脆声响在寂静高台上格外刺耳。周围同僚僵在原地，神情惊惧，彼此交换眼神却无人敢开口，空气像瞬间凝住一样。镜头先以中近景贴着她的侧脸和手部，捕捉她失神、屏息、猛然转身的瞬间，再顺着她的动作向后轻跟，看到她顾不上收拾算筹，提裙疾步冲下石阶，晨雾与衣角一起被带得翻起，整段气息从静到急，压抑中突然爆开。  

**声音设计**：  
前半段以极轻的晨风、衣料摩擦、算筹拨动木案的细碎声维持高处清寂；消息传开后的变化不依赖明确报信台词，而通过同僚间骤停的动作、压低的窃息、某个未说出口却已人人知晓的瞬间来制造断裂；算筹坠地、珠子滚落石阶的连续脆响成为本段主导声，余声在空台上拉长，压过一切人声。

## P02 书房门前的沉默交易

- 人物：顾星澜、守门内侍
- 场景：太史局内院、沈钧白书房门外
- 镜头组：单镜头
- 时长建议：10s

**导演阐述**：  
太史局内院幽深冷寂，廊下宫灯将灭未灭，师父书房门外立着一名年约四旬的内侍，垂眼守门，袖手而立，姿态恭顺却透着不容逾越的僵硬。顾星澜快步赶来，呼吸尚未平复，却在门前硬生生收住脚步，只微微移笏半寸，克制地行礼，压低声音请求入内取走历算旧物；内侍不看她，只用极低的声音点出“手令已下”，那一瞬两人都知道，眼前不是一道寻常的门，而是一道随时会将人卷入风波的界线。镜头以斜侧中景包住两人，缓缓推近，重点看内侍喉结轻动、顾星澜指节绷紧的细小动作，让对话像锋面贴着皮肤滑过。内侍沉默片刻后侧开半步，借口去净房，只给她“一炷香时间”，这一让步不是明说的帮助，而是久在宫中之人留出的余地，冷灰色廊庑和将熄的灯火把这场沉默交易照得格外险。  

**声音设计**：  
主声为长廊空响、将熄宫灯的轻微噼啪和两人压到最低的呼吸声；顾星澜疾行而至时鞋底叩地声偏急，停下后声音戛然而止；内侍说话要轻、短、低，不抢画面，却像把空间压得更窄；结尾以内侍离步前衣袍拂动和远处漏刻声隐约入场，提示时间开始流失。

## P03 暗格启卷

- 人物：顾星澜
- 场景：沈钧白书房
- 镜头组：多镜头（3组）
- 时长建议：15s

**导演阐述**：  
第一组镜头先建立书房全貌：案几收拾得近乎刻板整齐，砚中墨迹未干，墙上的星图却被粗暴揭去大半，地上散落着撕碎的奏折，像有人在极短时间内仓促掩埋关键痕迹。顾星澜立在门内，眼神从案头、墙面、地上的碎纸快速扫过，呼吸渐沉，她立刻意识到师父是在被带走前主动毁去重要文书。第二组镜头切到她蹲身搜寻，手指沿着书架底部和地面青砖一寸寸摸索，动作快但不乱，指腹沾上灰尘，青砖冰凉，她忽然在一处微微松动的砖缝前停住，整个人屏息，把那块砖轻轻撬起。第三组镜头推到暗格内部，一卷包着油纸的文稿安静躺着，几乎不染尘埃，像是被专门留给她的最后一句话；她展开油纸，看见“星澜亲启”四字时，眼神先是一颤，再强行压住情绪。镜头之间以硬切推进，每一切都比上一段更近、更紧，光线是从窗纸渗入的惨白天光，照出木案边缘和她指尖的微颤，让整段有一种在死寂中逐层逼近秘密核心的紧迫感。  

**声音设计**：  
书房内尽量收空，只保留门轴轻响、脚步压过碎纸的沙沙声、手指拂过木架与砖缝的摩擦声；撬砖时给一个细小却清晰的砖石松动声，形成听觉上的“找到”；油纸展开要有干涩、脆薄的质感，和她一瞬停住的呼吸一起构成段落落点。

## P04 税簿中的真相与暗号

- 人物：顾星澜
- 场景：沈钧白书房
- 镜头组：多镜头（4组）
- 时长建议：15s

**导演阐述**：  
顾星澜快速翻阅文稿，镜头贴近纸页上的字迹和她的眼睛交替切换，先看到“为师若有不测，此卷可证清白”，再落到税簿副本上被朱笔重重圈出的“灾年减税”字样，信息一层层压下来。她读到“税簿所减之数，恰与兵部军饷缺口相合”时，手指不自觉收紧纸页边缘，指节发白，呼吸明显乱了一拍；这不是模糊的账目异常，而是能直接咬合到军资去向的关键节点。随后镜头捕捉她注意到页角那些细小而有规律的墨点，她目光一凝，立刻取出算筹，在地上按二十四节气顺序排布。第三组镜头重点可视化她的推理过程：先给俯拍，看到她将纸页边角墨点依次对应到立春、雨水、惊蛰、春分等节气位置，再给手部近景，看她用木筹逐一点位、停顿、修正，最后把几处对应位置连成线，隐藏信息才逐步显形。第四组镜头给她一个近景，她在完整读出暗号内容——“谢时行经手，萧景琛授意，户部-兵部转账，银入私库”——后，猛地倒吸一口凉气，却没有发出声，震惊被死死压在胸腔里，只剩眼底骤然亮起的冷意；这一段节奏从阅读到破解越来越快，切换以硬切为主，像逻辑在她脑中飞速咬合，书房里只有翻页、筹珠碰地和她压抑的呼吸声，真相因此显得更尖锐。  

**声音设计**：  
翻页声要连续而偏急，前半段与她略失稳的呼吸叠加；算筹落地、移动、轻碰的声音承担“推理节拍器”功能，随着节气排序逐步加密；在她完成最后连线、读出完整暗号时短暂抽空环境声，只留一口吸气，让观众和角色同时抵达结论。

## P05 门外脚步，藏卷出室

- 人物：顾星澜、守门内侍
- 场景：沈钧白书房门外、太史局内院
- 镜头组：单镜头
- 时长建议：8s

**导演阐述**：  
就在顾星澜记下墨点位置、收好税簿副本时，门外忽然传来由远及近的脚步声，短促而稳定，提醒她一炷香将尽。她没有慌乱，只是迅速把文稿贴身收入袖中，整理神色，抚平衣襟，像把所有翻涌情绪硬压回骨头里，然后拉门而出。镜头从室内跟到门外，先扫过她恢复平静的面容，再转向那名归来的内侍；她郑重一礼，道谢时语气极轻，却是真心在谢一条余地。内侍摆手，低声点出“外头风紧”，话未说完便化作一声叹息转身离去，他的背影在长廊昏光里显得格外老迈。整段镜头不切，靠人物进退和视线完成张力，光线依旧偏冷，门内门外像两个世界，她从秘密中走出来时，已经不再是那个只会惊惶奔跑的弟子。  

**声音设计**：  
门外脚步要先远后近，成为整段的倒计时；顾星澜收卷、理衣、拉门的动作声要干净利落；她与内侍的对话都压低处理，重点保留一句未尽的话和末尾那声轻叹，让沉默比明说更有分量。

## P06 算房复盘，三日之限

- 人物：顾星澜
- 场景：顾星澜算房、殿上奏对闪回场景
- 镜头组：多镜头（3组）
- 时长建议：15s

**导演阐述**：  
她回到自己那间最简陋的算房，室内只有一案一榻，墙上挂着自绘星图，空间狭小却清洁克制，像她这个人一样，把锋芒都藏在秩序里。第一组镜头看她俯身把文稿藏入榻下暗格，再把税簿副本摊在案上细读，烛火微摇，映得她脸色冷白；镜头慢慢推近到纸面与她写下的时间线，清楚落出“今日初三”与“仅余三日”几个字，让当下的时间压力先稳稳立住。第二组镜头在她凝神停笔的一瞬，以短促闪回式硬切插入上月殿上奏对的残片——沈钧白于殿中直陈“今岁恐有西北边患，当备军资”，话音刚落，殿上气氛骤冷；高处随即压下一句“妖言惑众”，把这番预警当场钉死。闪回不做展开，只取这句奏陈、这句定性、群臣不敢作声的呼吸停顿与朝堂高处沉沉压下的威势，让观众立刻明白师父此前已因触碰军资与边患之事而被盯上。第三组镜头回到现实，她在纸上继续补齐账目、奏对、被带走三者之间的前后关系，字迹越来越快却始终工整，最后停在“要在三日内找到足够凭据”这一句上，让目标与期限同时落定；这一段的重心不是一次讲完全部信息，而是先立“只剩三日”的迫近，再用简短闪回补足她为何必须立刻行动。  

**声音设计**：  
以单盏烛火轻噼啪、笔尖划纸和她逐渐变重的呼吸作为现实层主声；写到“今日初三”“仅余三日”时，纸笔声要更清晰，像把字刻进空气里；闪回段加入殿中空阔回响、群臣衣袍摩擦、短促而有压迫感的人声片段，尤其突出“今岁恐有西北边患，当备军资”与“妖言惑众”这两句的撞击，再迅速抽回算房静室，形成现实与记忆的撞击。

## P07 夜观历法，谋局成形

- 人物：顾星澜
- 场景：顾星澜算房
- 镜头组：多镜头（4组）
- 时长建议：15s

**导演阐述**：  
夜色渐深，窗外初三的月亮被乌云遮去半边，天幕晦暗不明，顾星澜站在窗前望出去，耳边像回响起师父那句“天时可测，人事可为”。第一组镜头先给窗格外的云遮残月，再回到她沉静而发亮的眼睛，意识开始从哀恸转为筹算；这里强调的不是凭异象决断，而是她作为太史局中人对历法、时辰与礼制安排的长期训练。第二组镜头看她取纸执笔，迅速推演初六天象变化与朝仪时段的重合关系，笔锋在纸上连连落下，把“申正三刻”圈出。第三组镜头切到她翻开宫中起居注副本，指尖顺着“某人每日申初离开政事堂回府”的记录滑过，她的目光随之停住，思路在纸页与脑海间骤然贯通——若要在大朝会上提出议题，就必须借礼制程序与固定时辰把关键人物留在殿中。第四组镜头给她坐回案前，写下“时辰守门局”与两项条件：一是须有人在大朝会上当众提出账目问题，二是必须有铁证在手，而这份铁证不是泛泛账册，而是要查到户部与兵部银两转账记录；她想到一位可能愿意秉公发声的御史时，眼神终于有了明确落点。整段以几次干脆硬切完成思路跳跃，节奏由慢到快，烛光在纸上照出锋利阴影，像她正在把历法、礼制和人心一并纳入计划。  

**声音设计**：  
窗外远风、偶发檐滴与室内烛火声构成夜层底噪；她推演时以笔尖疾走、纸页翻动、算筹短促点案形成连续节拍；当她从起居注读到关键时辰、并把“户部与兵部银两转账记录”写入计划时，环境声微退，只留指尖滑过纸面的细声与一口极轻的吸气，突出思路贯通的瞬间。

## P08 焚纸立誓，拾珠开局

- 人物：顾星澜
- 场景：顾星澜算房
- 镜头组：多镜头（3组）
- 时长建议：10s

**导演阐述**：  
天将明时，顾星澜在纸上写下“三日之内，必救师父”，字不大，却一笔不抖，随后把纸条送入烛火，看它卷曲、发黑、化作纸灰飘散在晨光初起的屋内。她整理衣襟，收起所有情绪，准备像往常一样去司天台观测，外表必须平静得毫无破绽，只有观众知道她心里已经定下完整计划。镜头跟着她走到门边，又让她忽然停住，回身望见案旁那盒未收的算筹和散落的珠子——那是惊讯传来时留下的痕迹，也是这一局最初的起点。最后一组镜头俯拍她蹲下身，一颗一颗把珠子拾回算筹盒中，动作极稳，珠子落回盒内的声音清脆分明，像一步步落子的回响；晨光从门外慢慢漫入，冷色夜气退去，整段在安静中收束，带着一种不动声色却已经开始行动的决绝。  

**声音设计**：  
纸笔落字、火舌吞纸、灰烬轻坠构成前半段核心声；整衣、转身、停步都尽量克制，形成将出未出的停顿；尾声以一颗颗珠子落回盒中的清脆单音收束，每一声都像给计划校准一次节拍。

## 整体声音基调说明

- **总体原则**：本集少配乐或弱配乐，以环境音、器物声、纸张声、脚步声、呼吸声承担叙事推进，保持克制、真实、贴身。  
- **司天台段落（P01）**：以高处空寂和算筹坠地建立突发事件的第一层冲击，消息已传开的压力通过场域反应而非明确报信动作来体现。  
- **内院与书房段落（P02-P05）**：以低声对话、门轴、碎纸、油纸、算筹等近景拟音维持紧张，减少空泛静读感。  
- **算房推理段落（P06-P07）**：以笔划纸、烛火、翻页、算筹点案构成连续节拍，让信息整理和思路推进可听见、可感知；涉及军饷缺口、户部与兵部转账记录等关键信息时，声音要为信息落点让位。  
- **收束段落（P08）**：以焚纸和拾珠形成前后呼应，用清晰单一的声响完成情绪落点。
