# 人物设计技能

[技能说明]
    人物设计技能，将小说章节中的人物信息整理为可复用的人物设定资料。

    核心目标有两个：
    1. 从章节中识别出真正需要建立视觉设定的人物
    2. 根据章节情节，为人物补充当前章节可视化的穿着描述

    该技能输出的不是剧情总结，而是人物视觉设定资料，重点是人物长相基底、服饰状态和章节变化。

[文件结构]
    character-design-skill/
    ├── SKILL.md                                    # 本文件（技能流程说明）
    └── templates/
        └── character-design-template.md            # 人物设计产出模板

[执行流程]

    第一步：读取输入信息
        - 读取当前章节的小说正文
        - 读取用户提供的整体风格设定
        - 读取已有的 assets/character-design.md（如存在），保持人物前后统一

    第二步：提取本章人物
        从章节中筛选需要建立设定的人物，优先保留以下角色：
        - 本章有明确动作、对话或情节作用的人物
        - 在多章反复出现的主要角色
        - 虽然出场不多，但视觉辨识度很强、后续需要稳定表现的角色

        可跳过的人物：
        - 纯背景群演
        - 只被一笔带过、没有必要建立单独视觉设定的角色

    第三步：整理人物视觉基底
        为每个人物提炼稳定的基础设定：
        - 人物名
        - 身份/人物关系
        - 年龄段和性别
        - 脸型、五官、发型、体型等稳定外观信息
        - 整体气质

        如果原文没有写清全部细节：
        - 只能做合理补全
        - 补全要服从人物身份、时代、世界观和文风
        - 不得凭空加入强设定

    第四步：整理章节穿着
        根据本章剧情和出场环境，提炼人物在本章的穿着描述：
        - 上装、下装或整套服饰
        - 主色和辅助色
        - 材质和版型
        - 鞋子
        - 配饰或随身物件
        - 衣物状态（整洁、凌乱、破损、沾水、染血等）

        穿着描述要求：
        - 以“大致描述”为准，忠于原文线索
        - 重点写对画面有帮助的信息
        - 不要为了华丽而过度扩写

    第五步：判断素材状态
        对每个人物标注状态：
        - 新增：第一次建立设定的人物
        - 复用：外观和穿着与已有设定基本一致
        - 变体：同一人物在本章有明确换装、受伤、成长、身份转变等变化

        说明：
        - 复用项可简写，不必重复大段描述
        - 新增和变体项需要完整写出

    第六步：输出
        按照 templates/character-design-template.md 的格式输出。

        推荐写入：
        - assets/character-design.md

        写入方式：
        - 首章创建文件
        - 后续章节按章节追加
        - 复用项可在章节下标注“沿用前设”

[写作原则]
    - 先抓稳定特征，再写本章变化
    - 服装必须服务于人物身份、情境和时代背景
    - 避免空泛形容词，例如“很好看”“很有气质”
    - 尽量把描述写到可以直接被角色设计或分镜引用的程度

[注意事项]
    - 如果章节对人物穿着没有明确描写，应基于人物身份和场景进行保守补全
    - 如果同一人物在本章出现多套服装，优先记录戏份最重或最具辨识度的一套
    - 如果人物在章节中有强烈状态变化，可单独标为“变体”
