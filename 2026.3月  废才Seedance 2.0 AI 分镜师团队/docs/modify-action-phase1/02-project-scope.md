# 项目范围

## 1. 本期范围内

- 建立 `docs/modify-action-phase1/` 文档包
- 建立独立的 `video_only_once_phase1/` 代码目录
- 将 `fenjing_program` 与 `zaomeng` 纳入统一工作区路径模型
- 提供统一 CLI 集成骨架
- 提供统一任务契约、gate 契约和路径安全能力
- 为后续阶段准备测试、问题、验收基线

## 2. 本期范围外

- 不在本阶段重写 `fenjing_program` 内部流水线
- 不在本阶段重写 `zaomeng` 浏览器执行器
- 不在本阶段完成全自动图片/视频生产闭环
- 不在本阶段完成 LangGraph 等复杂编排接入
- 不在本阶段完成 TTS、渲染和成片交付

## 3. 系统边界

### 3.1 本阶段系统负责

- 统一目录
- 统一入口
- 统一文档
- 统一测试基线
- 统一问题记录
- 统一阶段推进规则

### 3.2 两个子系统继续负责

- `fenjing_program`
  - 文本理解和结构化生产
- `zaomeng`
  - 浏览器自动化执行

## 4. 交付物

- 文档包
- 统一代码工作区
- CLI 集成骨架
- 测试代码
- 阶段总结和问题点

## 5. 阶段划分

### Phase 01

- 统一工程骨架
- 统一路径与契约
- 统一测试基线

### Phase 02

- 接入知识库和角色锚定入口
- 接入 image task 打包与回写入口

### Phase 03

- 接入 continuity、video、audio、render 入口
- 增强状态机与 gate router

### Phase 04+

- 自动重试、异常队列、批处理排产
- 逐步压缩人工介入次数

## 6. 本期完成定义

只有满足以下条件，Phase 01 才算完成：

- 文档齐备
- 代码目录齐备
- 测试齐备
- 阶段总结齐备
- 问题日志完成第一次更新
