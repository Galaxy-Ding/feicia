# 项目范围

## 1. 范围定义

本项目交付的是一套基于 OpenClaw 的浏览器自动化工作流，覆盖即梦网页端图片生成、图片下载与命名、评分筛选、前后帧管理、短视频生成和执行证据沉淀。

## 2. 本期范围内

- 复用人工首次登录后的浏览器 profile
- 自动进入即梦图片生成页面
- 自动填写提示词并发起图片生成
- 自动等待生成完成并下载图片到本地
- 自动重命名图片并写入元数据
- 把图片交由独立评分角色进行打分
- 根据评分阈值保留前后帧
- 自动进入视频生成流程并提交 0-5 秒任务
- 自动下载视频并保存任务记录
- 输出日志、问题记录、测试和验收证据

## 3. 本期范围外

- 自动注册或自动输入账号密码登录
- 绕过验证码、滑块、人机校验或风控机制
- 逆向即梦网页接口或抓包模拟私有 API
- 即梦平台以外的多平台统一调度
- 自动完成复杂创意提示词编写
- 基于模型自动完成最终业务审美判断

## 4. 系统边界

### 4.1 系统负责

- 浏览器控制
- 本地文件管理
- 执行日志
- 评分与筛选编排
- 失败恢复和人工接管提示

### 4.2 人工负责

- 首次登录
- 验证码和异常风控处理
- 评分规则调整
- 最终抽检和正式验收

## 5. 关键假设

- 团队可稳定访问即梦网页
- 即梦页面存在可操作的图片生成与视频生成入口
- OpenClaw 可调用 browser 能力与本地文件处理能力
- 评分角色可输出结构化评分结果
- 本地环境具备稳定下载目录和持久化存储空间
