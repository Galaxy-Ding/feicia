# 任务清单

## 1. 阶段划分

建议分 6 个阶段实施。

## 2. 阶段任务

### 阶段 1：需求冻结与样例确认

- 明确即梦目标页面 URL、页面版本和浏览器环境
- 明确图片生成参数、视频生成参数和命名规则
- 准备 3 组黄金样例任务
- 冻结评分维度与阈值

### 阶段 2：浏览器基础能力打通

- 配置 OpenClaw browser skill
- 验证人工首次登录 profile 可复用
- 打通图片生成页的定位、输入、提交
- 打通下载目录监听与下载完成判断

### 阶段 3：图片链路闭环

- 完成图片重命名与元数据写入
- 接入评分角色
- 完成通过/失败归档
- 实现前后帧选择规则

### 阶段 4：视频链路闭环

- 打通视频生成页定位
- 完成前后帧上传或引用
- 等待视频生成并下载
- 关联视频与前后帧元数据

### 阶段 5：异常恢复与日志

- 落地状态机
- 实现失败重试
- 实现人工接管提示
- 完善运行日志、错误日志、截图证据

### 阶段 6：测试与验收

- 执行冒烟测试、回归测试、异常测试
- 准备验收证据
- 输出最终验收报告

## 3. 优先级

### P0

- 登录态复用
- 图片页自动输入与生成
- 图片下载
- 规范命名
- 评分结果结构化

### P1

- 前后帧自动筛选
- 视频页自动生成
- 失败恢复

### P2

- 批量任务调度
- 回归截图比对
- 统计报表

## 4. 里程碑

1. M1：图片页自动生成并下载成功
2. M2：图片评分和筛选闭环完成
3. M3：前后帧驱动视频闭环完成
4. M4：测试通过并进入验收

## 5. 依赖关系

- 阶段 2 完成后才能进入阶段 3
- 阶段 3 前后帧筛选完成后才能进入阶段 4
- 阶段 5 需贯穿阶段 2 到阶段 4 持续补齐
