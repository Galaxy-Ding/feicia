# 测试计划

## 1. 测试目标

验证即梦浏览器自动化工作流在真实页面和本地环境中能够稳定完成图片生成、下载命名、评分筛选和短视频生成。

## 2. 测试范围

- 登录态校验
- 图片页自动操作
- 图片下载与命名
- 评分角色输出与筛选逻辑
- 前后帧管理
- 视频页自动操作
- 失败恢复和人工接管
- 日志与证据输出

## 3. 测试类型

### 3.1 冒烟测试

- 检查 profile 能否复用
- 检查是否能进入即梦生成页
- 检查是否能下载一张图片

### 3.2 功能测试

- 覆盖每个功能需求点
- 检查命名规则与目录落盘
- 检查评分和筛选逻辑

### 3.3 异常测试

- 登录失效
- 选择器失效
- 下载超时
- 视频生成超时
- 评分返回非法 JSON

### 3.4 回归测试

- 选择器配置更新后重跑黄金样例
- 评分阈值更新后重跑筛选逻辑
- 下载目录变更后重跑命名链路

## 4. 测试环境

- 操作系统：Windows
- 浏览器：与 OpenClaw 兼容的目标浏览器
- 网络：可访问即梦网页
- 本地目录：具有读写下载权限
- 测试账号：已人工登录并确认登录态有效

## 5. 测试数据

建议准备：

- 3 个图片黄金样例任务
- 2 个评分边界样例
- 2 个视频生成样例
- 2 个异常样例

## 6. 关键测试用例

| 用例编号 | 名称 | 前置条件 | 预期结果 |
|---|---|---|---|
| TC-001 | 登录态校验通过 | 浏览器 profile 有效 | 系统识别为已登录并进入任务流程 |
| TC-002 | 图片生成成功 | 提示词有效 | 图片成功生成并可下载 |
| TC-003 | 图片下载与命名 | 图片已完成 | 文件落入规范目录且命名正确 |
| TC-004 | 评分通过筛选 | 图片质量满足阈值 | 结果进入 approved 并标记前帧或后帧 |
| TC-005 | 评分失败筛选 | 图片质量不满足阈值 | 结果进入 rejected 并写入失败原因 |
| TC-006 | 视频生成成功 | 前后帧齐备 | 成功下载视频并写入元数据 |
| TC-007 | 登录失效拦截 | profile 失效 | 任务转 BLOCKED，不继续执行 |
| TC-008 | 选择器失败恢复 | 页面局部改版 | 触发后备选择器或报错转人工 |

## 7. 通过准则

- P0 用例全部通过
- 高严重级别问题为 0 个未关闭
- 黄金样例至少连续通过 3 轮
- 验收证据完整可追溯
