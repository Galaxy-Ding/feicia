在视频演示的工作流基础上，我们还可以为 Skill 添加 Reference 参考文档，让 Claude 在执行任务时能够参考专业规范。以 iOS 开发为例，设计师 Skill 可以引用 Apple 的 Human Interface Guidelines (HIG) 作为参考文档。

project/

│       ├── product_manager_expert/
│       │   └── SKILL.md                    # 产品经理技能
│       ├── designer_expert_ios/
│       │   ├── SKILL.md                    # iOS 设计师技能

│           └── SKILL.md                    # 开发者技能
