# 人物设计产出模板

输出文件：
- assets/character-design.md

---

## 推荐格式

```markdown
# 人物设定库

## 第01章

### [人物名]（新增 / 复用 / 变体）

**身份**：[人物身份、职业、关系]

**人物基底**：
[稳定外观描述。包括年龄段、脸型、五官、发型、体型、整体气质。]

**本章穿着**：
[根据章节内容整理的大致服装描述。包括服装款式、颜色、材质、鞋子、配饰、服装状态。]

**补充说明**：
[如与前章一致可写“沿用前设”；如有变体可写明变化原因。]

---

### [人物名2]（新增）

**身份**：...

**人物基底**：...

**本章穿着**：...

**补充说明**：...

---

## 第02章

### [人物名]（变体）

**身份**：...

**人物基底**：
[可简写为“沿用前设”，如果外观基底未变。]

**本章穿着**：
[写明本章与前章的造型差异。]

**补充说明**：
[如“夜战状态”“便装版”“受伤后版本”等。]
```

---

## 输出规范

- 使用中文
- 以章节为单位整理
- 先写新增人物，再写变体人物
- 复用项可以简写，但不能影响后续人员理解
- 重点是“人物是谁、长什么样、本章穿什么”
